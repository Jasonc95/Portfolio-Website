About Me

Hello there, I'm Jason Clough, a beginner front-end developer from England. I'm excited to be starting my journey in the world of web development and learning as much as I can along the way.




I discovered my love for coding when I started creating basic websites for myself and friends. I soon realized the endless possibilities that coding could offer and decided to pursue it further. Although I'm a beginner, I'm eager to learn and have been actively improving my skills through online courses, Youtube and personal projects.

I'm also a big fan of version control systems like Git and GitHub, which allow me to collaborate with others and keep my code organized.
